## Description

DeveloperC++ is an educational platform that provides in-depth resources for mastering C++ programming. It covers everything from fundamental concepts to advanced algorithms, the history of the language, and its real-world applications in various industries.

## Features

- **Algorithms Section**: Comprehensive collection of C++ algorithm implementations
- **Concepts Guide**: In-depth explanations of C++ programming concepts and fundamentals
- **History Timeline**: Interactive timeline showcasing the evolution of C++ from its origins to modern day
- **Industry Applications**: Real-world case studies and use cases of C++ in different sectors
- **Code Examples**: Interactive code blocks with syntax highlighting
- **Comparison Tables**: Side-by-side comparisons of different C++ approaches and implementations

## Tech Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **UI Components**: 
  - Radix UI (headless components)
  - Material UI (MUI)
  - Shadcn/ui components
- **Animation**: Motion (Framer Motion)
- **Charts**: Recharts
- **Routing**: React Router

## Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Usage

After installation, run the development server and open your browser at `http://localhost:5173` to view the application.
